# DESCRIPTION

Installs [grc](http://korpus.juls.savba.sk/~garabik/software/grc.html) "Generic Colouriser" to add "stylesheets" and personal color settings for your favorite terminal apps.


# REQUIREMENTS

## Supported Platforms

The following platforms are supported by this cookbook, meaning that the recipes run on these platforms without error:

* Ubuntu
* Debian

# RECIPES

* `grc` - The default recipe.

# USAGE

This cookbook installs the grc components if not present, and pulls updates if they are installed on the system.
